## 科目一：理论基础
科目一的核心是**规律归纳与对比记忆相结合**。以下是重点整理的核心考点。

### 1. 记分新规重点：扣 3 分核心口诀 💡
以下是记 3 分的 8 个最常见交通违法行为速记口诀（对应驾考常考点）：

> **普逆电话高低速，**
> **不按车道故障灯。**
> **排队穿插惹人恼，**
> **三不让行记三分。**

**关键词详细拆解：**
* **普逆**：普通道路逆行（**注意**：高速/城市快速路逆行直接扣 12 分，所以普通路逆行扣 3 分）。
* **电话**：拨打、接听手持电话等妨碍安全驾驶的行为。
* **高低速**：在高速公路上行驶低于最低限速。
* **不按车道**：在高速公路或城市快速路上不按规定车道行驶。
* **故障灯**：车辆发生故障或事故后，不按规定使用灯光或设置警告标志（如未放三角牌、未开双闪）。
* **排队穿插**：前方车辆排队等候或缓慢行驶时，借道超车或者占用对面车道、穿插等候车辆。
* **三不让行**：
  1. 不按规定**让行**（如转弯车不让直行车）。
  2. 遇人行横道不按规定减速、停车、避让**行人**。
  3. 不按规定避让**校车**。

### 2. 扣分梯度对比表
为了防混淆，建议对比记忆相同违规行为在不同场景下的扣分区别：
| 违法行为 | 扣 12 分 | 扣 9 分 | 扣 6 分 | 扣 3 分 | 扣 1 分 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **高速逆行/倒车/掉头** | 🔴 极度危险（直接扣12） | - | - | - | - |
| **普通路逆行** | - | - | - | 🟡 扣 3 分 | - |
| **超速 (普通私家车)** | 高速/城快超50%以上 | 高速/城快超20%-50%<br>或普通路超50%以上 | - | 普通路超20%-50% | 普通路/高速超10%-20% |
| **不避让校车** | - | - | - | 🟡 扣 3 分 | - |
| **未礼让行人** | - | - | - | 🟡 扣 3 分 | - |
| **打电话/妨碍安全驾驶** | - | - | - | 🟡 扣 3 分 | - |

## 科目二：场地驾驶技能（五项）
重庆科目二目前最普遍的依然是**经典五项**（倒车入库、侧方停车、坡道定点起步与路边停车、直角转弯、曲线行驶）。
> **注意**：部分考场C2自动挡已取消坡道起步，具体以报考车型和考场要求为准。

### 通关核心：看准点位、控好车速
* **车速控制**：离合器是科目二的灵魂。要压住车速，越慢越好，但绝不能停顿（停顿可能会被系统判定为中途停车）。
* **坐姿与后视镜**：每次上车，**第一件事必须是调整座椅和后视镜**。点位完全取决于你的视线高度，一旦坐姿变了，平时练的参考点就全失效了。

## 科目三：道路驾驶技能
科目三是路考，核心在于**细节与安全意识**。重庆路考车流量普遍较大，对社会车辆的防御性驾驶和避让反应尤为重要。

### 必背细节顺口溜：
* **起步准备**：绕车一周看车底，开门上车系安全。调整镜子试离合，指纹验证别心急。
* **起步动作**：一踩（离合）二挂（一档）三转向（左转灯），四喇叭（按喇叭）五看（左后视镜）六手刹。
* **变道/超车**：必须先打转向灯，心中默数三秒整。偏头观察后视镜，安全平稳再动轮。

## 科目四：安全文明驾驶
科目四侧重于**安全防范意识和紧急情况处置**。
* **多选题做题技巧**：凡是带有“减速让行”、“安全避让”、“礼貌待人”、“慢行”字眼的，基本都是正确答案。
* **紧急情况处置**：爆胎时切忌猛踩刹车和急打方向，应当紧握方向盘，轻踩刹车减速，缓缓靠边。
